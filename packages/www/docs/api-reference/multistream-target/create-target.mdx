# `POST` create a multistream target

`POST /multistream/target` creates a `multistream target` object. The only
parameter you are required to set is the `url` of your target. The `name` will
be derived from the `url` host, if not specified.

Notice that only creating a `multistream target` is not enough for multistream
to start happening. You also need to reference the target from the `stream` that
you want to be multistreamed. Refer to
[Multistream](/docs/guides/start-live-streaming/multistream) guide for more
information.

Here is an example request:

#### curl

```bash
curl -X POST 'https://livepeer.com/api/multistream/target' \
-H 'content-type: application/json' \
-H 'authorization: Bearer {api-key}' \
-d '{
    "name": "My Multistream Target",
    "url": "rtmp://rtmp.streaming.tv/live/SECRET"
}'
```

#### JavaScript + axios

```javascript
const axios = require("axios");

axios
  .post("https://livepeer.com/api/multistream/target", {
    headers: {
      Authorization: "Bearer {api_key}",
      "Content-Type": "application/json",
    },
    data: JSON.strigify({
      name: "My Multistream Target",
      url: "rtmp://rtmp.streaming.tv/live/SECRET",
    }),
  })
  .then((res) => {
    console.log(JSON.stringify(res.data));
  })
  .catch((error) => console.log(error));
```
